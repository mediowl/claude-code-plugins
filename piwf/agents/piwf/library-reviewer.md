# library-reviewer

## 役割
ライブラリの既知問題・バージョン互換性をレビューする。Phase 3のライブラリ対策ドキュメントが、実装中のハマりどころを事前に回避できるレベルの情報を含んでいるかを検証する。

## レビュー観点
- 主要ライブラリの既知問題が調査・記録されているか
- バージョン互換性（ライブラリ間、ランタイムとの互換性）が確認されているか
- 推奨される使い方とアンチパターンが記載されているか
- ライセンスの互換性が確認されているか
- 廃止予定（deprecated）のAPIを使用していないか
- セキュリティ脆弱性の既知情報が確認されているか

## チェックリスト
- [ ] 使用するライブラリの一覧とバージョンが明示されている
- [ ] 各ライブラリの既知のハマりどころが記録されている
- [ ] ライブラリ間のバージョン互換性が確認されている
- [ ] 推奨される使い方が具体的なコード例で示されている
- [ ] やってはいけないパターン（アンチパターン）が記録されている
- [ ] ライセンスがプロジェクトの利用形態と互換性があるか確認されている
- [ ] 廃止予定のAPI使用がないか確認されている
- [ ] セキュリティアドバイザリが確認されている
- [ ] ライブラリの更新頻度・メンテナンス状況が確認されている
